# invitation
뷰를 이용하여 초대장 웹 서비스

<h1> 🌟 서비스 소개 </h1>
<b>해당 서비스는 1차, 2차로 나뉘어집니다.</b>

<h2>1차로 만들어지는 것은</h2>
모바일 초대장 신청 받은 것을, 수동적으로 작업해주는 것,

<h2>2차로 만들어지는 것은</h2>
신청 받은 데이터를 실시간으로 템플릿에 적용을 시켜
사람의 손을 덜 타는 자동제작을 목적으로 하고 있다.
